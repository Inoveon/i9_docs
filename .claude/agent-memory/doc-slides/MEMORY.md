# doc-slides — Memória Operacional

## Estruturas por Tipo

| Tipo | Slides | Estrutura |
|------|--------|-----------|
| Pitch Deck | 10-15 | Problema → Solução → Mercado → Tração → Produto → Modelo → Equipe → Roadmap → CTA |
| Deck Comercial | 8-12 | Cenário → Desafio → Solução → Features → Cases → Implementação → Investimento → CTA |
| Deck Executivo | 6-8 | Sumário → KPIs → Realizações → Desafios → Próximo Período → Necessidades |

## Regras de Ouro

- 1 ideia por slide
- Título = afirmação forte, nunca pergunta genérica
- Abertura com impacto (número, pergunta retórica, afirmação forte)
- Máx 7 bullets por slide — dividir em dois se passar
- Sempre terminar com CTA específico e mensurável

## Limites de Slides

- Pitch: 15 | Comercial: 12 | Executivo: 8 | Treinamento: 20
