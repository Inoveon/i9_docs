# Templates

Templates base para criação de novos documentos.

## 📄 Disponíveis

### apresentacao-template.html
Template para criar novas apresentações seguindo o padrão visual i9ON/Inoveon.

**Recursos:**
- Scroll-snap navigation
- Gradientes de marca
- Material Design Icons
- CSS variables customizáveis
- Slides responsivos

### relatorio-template.html
Template para criar relatórios técnicos e de auditoria.

**Recursos:**
- Estrutura de relatório padrão
- Seções pré-definidas
- Estilos para código e tabelas
- Layout profissional

## 🚀 Como Usar

1. Copie o template apropriado
2. Renomeie para `index.html` no destino
3. Customize o conteúdo
4. Atualize variáveis CSS se necessário
5. Teste localmente antes de commitar

## 📝 Boas Práticas

- Mantenha referências a assets compartilhados em `/assets/`
- Use caminhos relativos
- Documente customizações significativas
- Preserve estrutura base do template
