## 📝 Como Adicionar Nova Apresentação

1. Criar pasta com nome descritivo: `apresentacoes/nome-apresentacao/`
2. Adicionar arquivo `index.html`
3. Usar template de `templates/apresentacao-template.html`
4. Atualizar este README
