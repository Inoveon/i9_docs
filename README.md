# i9ON/Inoveon - Documentação Corporativa

Repositório centralizado de apresentações, relatórios, documentação técnica e agents.

## 📁 Estrutura

```
i9_docs/
├── apresentacoes/      # Apresentações corporativas e de investimento
├── relatorios/         # Relatórios de auditoria e análises
├── documentacao/       # Documentação técnica do ecossistema
├── propostas/          # Propostas comerciais e técnicas
├── agents/             # Agents e Skills (21 agents documentados)
├── assets/             # Recursos compartilhados (logos, estilos)
└── templates/          # Templates base para novos documentos
```

## 🚀 Visualizar Localmente

```bash

# Opção 2: Node.js
npx serve

# Opção 3: PHP
php -S localhost:8000
```

Acesse: `http://localhost:8000`

## 📝 Convenções

- Arquivos HTML principais usam nome `index.html`
- Assets compartilhados ficam em `/assets/`
- Cada seção possui seu próprio README.md
- Use caminhos relativos para assets compartilhados
