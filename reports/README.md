# Relatórios

Relatórios de auditoria, análises técnicas e diagnósticos do sistema.

## 📋 Disponíveis

### Auditoria 2025
**Arquivo:** `auditoria/2025/index.html`
**Descrição:** Relatório completo de auditoria do código e infraestrutura 2025.

### Auditoria Git
**Arquivo:** `auditoria/git/index.html`
**Descrição:** Análise detalhada do histórico de commits, padrões de desenvolvimento e qualidade do repositório.

## 📝 Como Adicionar Novo Relatório

1. Criar pasta por tipo e ano: `relatorios/[tipo]/[ano]/`
2. Adicionar arquivo `index.html`
3. Usar template de `templates/relatorio-template.html`
4. Atualizar este README
