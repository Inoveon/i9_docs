content_copy Como Usar: Copy-Paste dos Componentes

1 Abra o arquivo de componentes no Chrome

Navegue até: assets/graficos-exemplos.html ou assets/formularios-componentes.html

2 Encontre o componente que deseja

Role a página até encontrar o componente visual que você quer usar (ex: Card Comparativo, Gráfico de Pizza, Input, etc.)

3 Clique com botão direito → Inspecionar

Clique com o botão direito do mouse no componente e selecione "Inspecionar" (ou pressione F12)

4 Copie o código HTML

No DevTools, clique com botão direito no elemento HTML destacado → Copy → Copy outerHTML

5 Cole no seu projeto

Cole o código diretamente no seu arquivo HTML. O CSS está inline, então funciona imediatamente!

Exemplo Prático

Vamos copiar este card comparativo abaixo e usar no seu projeto:

Vendas Totais
R$ 127k
▲ 12.5% vs. mês anterior

arrow_upward Clique com botão direito neste card azul acima → Inspecionar → Copy outerHTML

Depois de copiar, cole assim no seu HTML:

<!DOCTYPE html> <html lang="pt-BR"> <head> <meta charset="UTF-8"> <title>Meu Dashboard</title> <!-- Não esqueça de incluir as fontes --> <link href="https://fonts.googleapis.com/css2?family=Inter:wght@300;400;500;600;700;800&display=swap" rel="stylesheet"> <link href="https://fonts.googleapis.com/icon?family=Material+Icons" rel="stylesheet"> </head> <body> <!-- COLE O COMPONENTE AQUI --> <div style="background: linear-gradient(135deg, #005BAA 0%, #011E4B 100%); border-radius: 16px; padding: 25px; color: white;"> ... </div> </body> </html>

lightbulb Dicas Importantes:

  • Sempre inclua as fontes (Inter e Material Icons) no <head>
  • O CSS está inline, então você não precisa de arquivo CSS separado
  • Customize as cores e valores editando os números e hexadecimais no código
  • Teste a responsividade redimensionando a janela do navegador
  • Para componentes complexos, copie toda a seção incluindo o container
