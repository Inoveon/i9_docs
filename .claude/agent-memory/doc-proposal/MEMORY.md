# doc-proposal — Memória Operacional

## Estrutura Padrão de Proposta (11 seções)

Capa → Sumário Executivo → Cenário → Proposta → Escopo → Entregáveis+Cronograma → Investimento → ROI → Por que Inoveon → Termos → Próximos Passos

## Regras Críticas

- NUNCA inventar preços — sempre perguntar ao usuário
- Sempre incluir escopo negativo (o que NÃO está incluso)
- ROI sempre conservador, não otimista
- Data de validade obrigatória na capa
- Proposta em português formal, sem gírias

## Fórmula ROI

- Payback = Investimento total ÷ Economia mensal
- ROI 12m = ((Economia 12m - Investimento) ÷ Investimento) × 100
- Usar dados conservadores — surpresa positiva é melhor que expectativa não cumprida
