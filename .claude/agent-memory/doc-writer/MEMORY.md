# doc-writer — Memória Operacional

## Voz da Marca Inoveon

- Tom: profissional, direto, orientado a dados, sem jargão desnecessário
- Sempre usar voz ativa
- Público primário: gestores, diretores e CEOs de varejo/food service
- Público secundário: equipes técnicas, operadores de PDV
- Frases proibidas: "sinergias", "ecossistema robusto", "solução de ponta", "estado da arte"

## Estrutura Padrão de Apresentação

1. Capa — título impactante + subtítulo contextualizador
2. Problema — 1 slide, máx 3 bullets
3. Solução — 1-2 slides, foco no diferencial
4. Como funciona — 2-3 slides, processo passo a passo
5. Resultados / Prova social — dados, cases, métricas
6. Investimento — tabela clara e objetiva
7. Próximos passos — CTA específico

## Estrutura Padrão de Proposta

1. Sumário Executivo → 2. Cenário → 3. Proposta → 4. Escopo → 5. Entregáveis → 6. Investimento → 7. ROI → 8. Próximos Passos

## Estrutura Padrão de Relatório

1. Sumário Executivo (3-5 bullets) → 2. KPIs → 3. Análise Detalhada → 4. Próximos Passos

## Regras de Escrita

- Máximo 7 bullets por lista
- Títulos: afirmação forte, nunca pergunta genérica
- Dados: sempre com fonte ou "dados internos Inoveon"
- CTA: específico com prazo quando possível
- Voz ativa obrigatória
